# Project Overview

The **RAG Candidate Screening System** is a web-based application that helps recruiters and HR teams quickly screen resumes against job descriptions using AI.

**Objectives:**
- Automate resume analysis
- Generate dense technical summaries of resumes
- Compute semantic similarity between candidate profiles and job descriptions
- Save recruiter time and increase hiring efficiency

**Core Technologies:**
- Python 3.10+
- Streamlit (Frontend / UI)
- ScaleDown API (Resume compression)
- Sentence-Transformers (`all-MiniLM-L6-v2`) for embeddings
- Numpy for similarity computation
